# 🌿 Greenden - Plant Store Website

Greenden is a modern, responsive plant store website built using **HTML5** and **Tailwind CSS**. It contains multiple pages such as Home, Products, and Contact with a clean UI and simple structure.

---

## 🚀 Pages Included

- 🏠 **index.html** → Home page (Landing page)
- 🪴 **product.html** → Products listing page
- 📞 **contact.html** → Contact form page
- 🖼️ **images/** → All project images

---

## ✨ Features

- Fully responsive design
- Modern UI using Tailwind CSS
- Multiple pages website
- Clean navigation bar
- Product showcase layout
- Contact form UI
- Image-based design
- Beginner-friendly structure

---

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS (CDN)
- Google Fonts (optional)
- Images (Unsplash / local assets)
